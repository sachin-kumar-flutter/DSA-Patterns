# BFS in Trees

Breadth First Search (BFS) is used for level-wise traversal of trees.

## 🔑 Pattern

- Use queue
- Process nodes level by level

---

## 📂 Problems

- Zigzag Level Order Traversal  
- Kth Largest Level Sum  

---

## ⚡ Key Takeaway

Use BFS when:
- You need level-wise processing  
- You need shortest distance in tree
