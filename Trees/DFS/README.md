# DFS in Trees

Depth First Search (DFS) is used for recursive traversal and subtree-based problems.

## 🔑 Pattern

- Use recursion
- Explore left and right subtrees

---

## 📂 Problems

- Maximum Product of Splitted Binary Tree  
- Subtree With All Deepest  
- Sum Root to Leaf Binary Numbers  

---

## ⚡ Key Takeaway

Use DFS when:
- Problem involves subtrees  
- You need bottom-up computation
