# 🌊 BFS & DFS Basics

> Learn to explore graphs and grids using two of the most fundamental traversal algorithms — **Breadth-First Search (BFS)** and **Depth-First Search (DFS)**.

This directory contains beginner-friendly graph and matrix traversal problems that build a strong foundation for advanced graph algorithms like Topological Sort, Dijkstra, Union-Find, and Shortest Path algorithms.

---

## 📚 Problems Covered

- 🏝️ Number of Islands
- 🌊 Pacific Atlantic Water Flow
- 🍊 Rotten Oranges

---

## 🎯 Concepts Practiced

- Graph Traversal
- Matrix Traversal
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- Multi-Source BFS
- Connected Components
- Flood Fill Technique
- Direction Arrays (`dx`, `dy`)
- Visited Array Optimization

---

## 💡 What You'll Learn

- When to choose **BFS** over **DFS**
- Converting matrix problems into graph problems
- Traversing all connected cells efficiently
- Handling multiple starting points
- Avoiding revisiting nodes
- Time and Space Complexity analysis

---

## 🚀 Why These Problems?

Most graph interview questions are simply variations of BFS or DFS.

Master these patterns once, and you'll be able to solve dozens of graph and grid problems with confidence.

> **"If you can traverse it, you can solve it."** 🚀
