# Monotonic Stack

A monotonic stack maintains elements in increasing or decreasing order.

## 🔑 Pattern

- Used to find next greater/smaller element
- Maintains sorted structure

---

## 📂 Problems

- Daily Temperatures  
- Largest Rectangle in Histogram  
- Maximal Rectangle  
- Remove K Digits  

---

## ⚡ Key Takeaway

Use monotonic stack when:
- You need nearest greater/smaller element
- You want O(n) optimization over brute force
