# Subsequence DP

Problems involving strings or arrays where we select or skip elements.

## 🔑 Patterns

- Longest Common Subsequence (LCS)
- String transformation
- Matching characters

## 🧠 Approach

- Define dp[i][j] based on prefixes
- If characters match → include
- Else → take best of skipping options

## ⭐ Problems

- Longest Common Subsequence
- Delete Operation for Two Strings
- Minimum ASCII Delete Sum
- Maximum Dot Product of Subsequences
