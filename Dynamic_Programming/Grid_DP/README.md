# Grid DP

Problems where movement is typically restricted to directions like right and down in a matrix.

## 🔑 Patterns

- Path counting
- Minimum / Maximum path sum
- State expansion (extra dimension like k operations)

## 🧠 Approach

- Define dp[i][j] as the answer at cell (i, j)
- Transition from top or left cell
- Handle constraints like obstacles or limited operations

## ⭐ Problems

- Paint N x 3 Grid
- Maximum Coins with Robber Neutralization
