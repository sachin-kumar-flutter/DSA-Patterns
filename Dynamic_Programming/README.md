# Dynamic Programming

This section contains solutions to classic and advanced Dynamic Programming problems categorized by patterns.

## 📌 Topics Covered

- Grid DP
- Subsequence DP
- State-based DP

## 🧠 Key Concepts

- Overlapping subproblems
- Optimal substructure
- State definition and transitions
- Memoization vs Tabulation

## 📂 Structure

- **Grid-DP** → Problems involving 2D grids and path-based transitions  
- **Subsequence-DP** → Problems involving sequences and string transformations  

## 🚀 Highlight Problems

- Maximum Coins with Robber Neutralization (3D DP)
- Longest Common Subsequence
- Maximum Dot Product of Subsequences
