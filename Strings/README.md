# 🧵 Strings - Problem Solving Patterns

This directory contains problems based on **string manipulation**, focusing on pattern recognition, parsing, and traversal techniques.

---

## 📌 Covered Patterns

- 🔹 String Parsing
- 🔹 Simulation
- 🔹 Matrix Traversal using Strings
- 🔹 Character Transformation & Formatting

---

## 📂 Problems

### 1. Mask Personal Information
- 📄 File: `Mask_Personal_Information.cpp`
- 🧠 Pattern: String Parsing + Formatting
- 💡 Idea:
  - Identify whether input is email or phone number
  - Apply masking rules accordingly
  - Handle edge cases like country codes

---

### 2. Decode the Slanted Ciphertext
- 📄 File: `decode_ciphertext.cpp`
- 🧠 Pattern: Matrix Diagonal Traversal + Simulation
- 💡 Idea:
  - Convert string into 2D grid (row-wise)
  - Traverse diagonally to decode message
  - Remove trailing spaces

---

## 🚀 Key Takeaways

- Strings often combine with other concepts like **arrays and matrices**
- Many problems are not purely string-based but involve **simulation**
- Focus on:
  - Traversal direction
  - Edge case handling
  - Clean string building

---

## 🧠 Tip

> If a string problem feels unusual, try visualizing it as a **2D structure or simulation problem**.
