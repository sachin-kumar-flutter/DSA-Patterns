# Inplace Marking

This section covers problems where the input array is modified to store additional information without using extra space.

## 🧠 Key Idea

Use the array itself as a hash map by marking indices.

---

## 🔑 Pattern

- Use indices to represent values
- Mark visited elements using negative sign or offset

---

## 📂 Problems

### Find All Disappeared Numbers
- Pattern: Inplace Marking
- Idea: Mark visited indices using negative values to identify missing numbers

---

## 🎯 Why Important?

- Saves space (O(1) extra space)
- Common interview pattern
- Improves understanding of index-based tricks
