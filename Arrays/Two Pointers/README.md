# 👣 Two Pointer Patterns

Welcome to the **Two Pointer Pattern** section 🚀
One of the most powerful and frequently used techniques in DSA.

---

## 🧠 What is Two Pointer Technique?

The idea is simple:

> Use two indices to traverse data efficiently instead of nested loops.

This helps reduce time complexity from **O(n²) → O(n)** in many problems ⚡

---

## 🎯 When to Use?

✅ Sorted arrays
✅ Pair finding problems
✅ Merging / intersection problems
✅ Removing duplicates
✅ Sliding window variations

---

## 🔥 Problem Covered

### 📌 Intersection of Two Sorted Arrays

* Use two pointers to traverse both arrays
* Compare elements and move pointers smartly
* Skip duplicates to maintain distinct results

---

## ⚙️ Key Insight

Since arrays are sorted, we don’t need extra space like hash sets.
We can solve it efficiently in a single pass 💡

---

## ⏱️ Complexity

* **Time:** O(n + m)
* **Space:** O(1) (excluding output)

---

This pattern appears again and again in interviews:

✨ Remove Duplicates
