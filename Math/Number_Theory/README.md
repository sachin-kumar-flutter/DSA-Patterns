# Number Theory

This section contains problems related to prime numbers and mathematical properties.

## 🧠 Key Concepts

- Prime number generation
- Sieve of Eratosthenes
- Factorization

## 📂 Problems

### Closest Prime Pair
- Pattern: Sieve + Greedy Search
- Idea: Find nearest primes in range

### Count Primes (Sieve)
- Pattern: Sieve of Eratosthenes
- Idea: Efficiently generate primes

### Factorial Trailing Zeroes
- Pattern: Mathematical Observation
- Idea: Count factors of 5 in factorial

---

## ⚡ Key Takeaway

- Use sieve for prime problems
- Avoid brute force for large numbers
