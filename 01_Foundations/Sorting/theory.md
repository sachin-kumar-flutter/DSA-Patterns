# 🔢 Sorting Algorithms

## 📌 What is Sorting?

Sorting is the process of arranging elements in a specific order (usually ascending or descending).

👉 Think of it as organizing data so that searching and processing become faster and easier.

---

## ⚡ Why is Sorting Important?

* Improves efficiency of algorithms
* Enables binary search
* Helps in solving many pattern-based problems
* Foundation for advanced techniques

---

## 🧠 Types of Sorting Algorithms

### 🔹 Simple Sorting (Brute Force)

* Bubble Sort
* Selection Sort
* Insertion Sort

👉 Easy to understand, but not efficient for large data

---

### 🔹 Efficient Sorting (Divide & Conquer)

* Merge Sort
* Quick Sort

👉 Much faster and widely used in practice

---

### 🔹 Non-Comparison Sorting

* Counting Sort
* Radix Sort
* Bucket Sort

👉 Used when constraints allow optimization

---

## ⏱️ Time Complexities

| Algorithm      | Best       | Average    | Worst      | Space    |
| -------------- | ---------- | ---------- | ---------- | -------- |
| Bubble Sort    | O(n)       | O(n²)      | O(n²)      | O(1)     |
| Selection Sort | O(n²)      | O(n²)      | O(n²)      | O(1)     |
| Insertion Sort | O(n)       | O(n²)      | O(n²)      | O(1)     |
| Merge Sort     | O(n log n) | O(n log n) | O(n log n) | O(n)     |
| Quick Sort     | O(n log n) | O(n log n) | O(n²)      | O(log n) |

---

## 🎯 When to Use What?

* Small data → Insertion Sort
* Stable sorting needed → Merge Sort
* Fast average case → Quick Sort
* Known range → Counting Sort

---

## 🔥 Important Concepts

* Stability of sorting
* In-place vs out-of-place
* Divide and Conquer
* Partitioning (Quick Sort)

---

## ⚠️ Common Pitfalls

* Choosing wrong sorting for constraints
* Ignoring worst-case in Quick Sort
* Not understanding stability

---

## 💡 Key Insight

Sorting is not just about ordering data—it is often the **first step to simplify complex problems**.

👉 Many problems become easier once the array is sorted.

---

## 🧾 Notes

Sorting algorithms are fundamental to computer science.
A strong understanding of sorting helps in mastering patterns like binary search, greedy, and divide & conquer.
