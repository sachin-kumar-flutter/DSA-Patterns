# Arrays

## 📌 What is an Array?

An array is a collection of elements stored in **contiguous memory locations**, allowing efficient access using an index.

---

## ⚡ Key Properties

* Fixed size (in static arrays)
* Elements are stored in contiguous memory
* Index-based access (0-based indexing in C++)

---

## ⏱️ Time Complexities

| Operation | Complexity |
| --------- | ---------- |
| Access    | O(1)       |
| Search    | O(n)       |
| Insertion | O(n)       |
| Deletion  | O(n)       |

---

## 🧠 Types of Arrays

* 1D Array
* 2D Array (Matrix)
* Dynamic Array (vector in C++)

---

## 📍 Memory Representation

Address of element:

```
Address(arr[i]) = Base + i * size
```

---

## 🎯 When to Use Arrays

* When fast access is required
* When data size is fixed or known
* When working with contiguous data (prefix sums, sliding window)

---

## ⚠️ Limitations

* Fixed size (in static arrays)
* Costly insertion/deletion
* Memory wastage if not used fully

---

## 🔥 Important Patterns Related to Arrays

* Prefix Sum
* Sliding Window
* Two Pointers
* Kadane’s Algorithm

---

## 🧾 Notes

Arrays are the foundation of most problems. Strong understanding of arrays leads to easier mastery of advanced patterns.
