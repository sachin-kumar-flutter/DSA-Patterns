# Linked List

This section contains problems based on singly linked list operations and pointer manipulation

## 🧠 Key Concepts

- Pointer manipulation
- Reversal techniques
- Node re-linking
- Traversal and modification

---

## 📂 Problems

### 1. Add Two Numbers
- Pattern: Linked List + Simulation
- Idea: Simulate addition digit by digit using carry

### 2. Odd Even Linked List
- Pattern: Pointer Rearrangement
- Idea: Separate odd and even indexed nodes

### 3. Reverse Linked List
- Pattern: Iterative / Recursive Reversal
- Idea: Reverse pointers to change direction of list

---

## 🎯 Why Linked List?

- Tests pointer handling skills  
- Common interview topic  
- Requires careful implementation  

---

## ⚡ Key Takeaway

Focus on:
- Pointer updates  
- Edge cases (null, single node)  
- In-place operations
