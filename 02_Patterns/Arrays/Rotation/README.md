## 🧠 Why Rotation Matters?

Array rotation is not just a problem — it’s a **pattern** that appears in:

* Cyclic shifts 🔁
* String manipulation 🧵
* Queue simulations 📦
* Real-world systems (circular buffers, scheduling)

If you understand rotation deeply, you unlock a whole class of problems.

---

## ⚙️ Approaches Covered

### 1️⃣ Reversal Algorithm (Most Important ⭐)

* Reverse first part
* Reverse second part
* Reverse whole array

💡 Clean, intuitive, and interview-favorite.

⏱️ Time: `O(n)`
💾 Space: `O(1)`

---

### 2️⃣ Juggling Algorithm (GCD Based 🔄)

* Divide array into cycles using GCD
* Rotate elements within each cycle

💡 Slightly tricky, but shows deeper understanding.

⏱️ Time: `O(n)`
💾 Space: `O(1)`
